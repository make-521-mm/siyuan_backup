## 🛠️ 核心功能

- **🔒 AES-256 加密压缩**：使用强密码保护你的笔记数据，防止云端泄露
- **☁️ WebDAV 自动同步**：将备份文件自动移动到挂载的WebDAV网盘目录
- **🧹 智能清理**：仅保留最新的N个备份，自动删除旧文件以节省空间
- **📝 详细日志**：记录运行时间、压缩耗时、传输耗时及错误信息

## ⚙️ 快速开始

### 1. 环境准备

确保你的电脑上已安装 **Python 3.7** 或更高版本。

### 2. 安装依赖

在脚本所在目录打开终端（CMD/PowerShell），运行以下命令安装压缩库：

```
pip install py7zr
```

### 3. 配置脚本

请使用文本编辑器打开 `siyuan_back.py`，修改第一部分的配置参数：

```
# --- 1. 配置路径 ---
SIYUAN_DATA_PATH = r"你的\思源\data\目录"  # 例如: D:\SiYuan\data
WEBDAV_DIR = r"你的\WebDAV\挂载盘\目录"     # 例如: Z:\Backup\
LOCAL_TEMP_DIR = r"本地\临时\SSD\目录"      # 建议使用SSD，速度快
BACKUP_PASSWORD = "你的加密密码"            # 请务必修改此密码！
```

### 4. 运行脚本

双击运行 `siyuan_back.py`，或者在终端中执行：

```
python siyuan_back.py
```

## 📂 输出说明

运行后，你将在指定目录看到以下内容：

1. **WebDAV目录**：生成类似 `SiYuan_data_20231010_120000.7z` 的加密压缩包
2. **脚本同级目录**：生成 `backup.log` 日志文件（用于排查双击运行时的黑屏报错）

## 🗑️ 清理策略

默认保留 **5** 个最新的备份文件。如果需要修改保留数量，请调整 `KEEP_COUNT` 常量。

## 📝 排查问题

如果备份失败，请检查脚本所在目录下的 `backup.log` 文件，通常错误原因为：

- 路径中包含中文或空格（建议路径全英文）
- WebDAV盘符未挂载成功（Z盘断开）
- 杀毒软件锁定了文件
