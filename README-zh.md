# 📁 思源笔记 (SiYuan) 自动化加密备份脚本

本脚本用于自动化备份 [思源笔记](https://b3log.org/siyuan/) 的数据目录。它支持 **7z 加密压缩**、**WebDAV 自动上传** 以及 **旧备份自动清理** 功能。脚本运行时会同时输出日志到控制台和本地文件，方便排查后台运行时的错误。

## 🛠️ 核心功能

-   **🔒 AES-256 加密压缩**：使用强密码保护你的笔记数据，防止云端泄露。
-   **☁️ WebDAV 自动同步**：将备份文件自动移动到挂载的 WebDAV 网盘目录。
-   **🧹 智能清理**：仅保留最新的 N 个备份，自动删除旧文件以节省空间。
-   **📝 详细日志**：记录运行时间、压缩耗时、传输耗时及错误信息。
